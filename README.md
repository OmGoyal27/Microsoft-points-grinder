Hello everyone! This project is for grinding the microsoft reward points quicker, both for phone and desktop.
The main.py file is for desktop mode. It opens a new tab and writes off some words.
The mobile.py file clicks on a location on the screen and starts typing, hit enter, wait for some time and repeat it again.
The add.py file helps you to add words in the words.json file easily. Type save to load all of those words into the file.
the add_start file helps you to add words in the start.json file easily. Type save to load all of those words into the file.
The add_meaning.py file helps you add words for search of meaning of words. It is only available for mobile.py file.
The printt.py file prints all the words in the words.json file.
